# 부모-자녀 간 용돈 관리 및 자녀의 경제관념 확립 서비스

## 프로젝트 개요
- **프로젝트명:** 키득키득(KDKD)
- **프로젝트 기간:** 2025.08.25 ~ 2025.09.29 
- **프로젝트 형태:** 팀 프로젝트 (핀테크 + AI 융합 서비스)  
- **목표:** 부모와 자녀가 함께하는 안전하고 체계적인 용돈 관리 및 경제 교육 지원
- **주요 타겟 사용자:** 자녀의 금융 습관 형성을 돕고자 하는 부모와 디지털 금융을 처음 경험하는 어린이·청소년

## 프로젝트 개요

1. **부모-자녀 계정 연동**

   - 부모는 자녀의 경제 활동을 모니터링하면서도 직접 개입 없이 관리 가능

2. **용돈 관리 및 자동이체**

   - 정기적/일회성 용돈 지급 및 자동 이체 기능을 통한 계획적 금전 관리

3. **저축(모으기 통장)**

   - 자녀가 목표를 설정하고 저축을 통해 성취감을 느낄 수 있는 구조 설계

4. **SSAFY 금융망 기반 가상 카드 발급**

   - 부모 명의 카드를 직접 사용하지 않고, 자녀 전용 가상 카드로 안전한 결제 제공

5. **미션/보상 시스템**

   - 부모가 설정한 미션(집안일, 학습 목표 등)을 달성하면 용돈·보상 지급

6. **소비/저축 내역 시각화**

   - 카드·그래프 UI 기반으로 소비 및 저축 내역을 직관적으로 확인

7. **캐릭터 키우기 (키덕이)**
   - 자녀의 흥미를 유발하는 아바타 성장 시스템
